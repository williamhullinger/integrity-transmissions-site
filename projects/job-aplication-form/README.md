# Job Application Form

A responsive job application form built as part of the **FreeCodeCamp (FCC)** curriculum.  
This project demonstrates proper HTML form structure, accessibility best practices, and CSS pseudo-classes for validation and interactivity.

---

## 📌 Project Overview

This project is a simple job application form that allows users to:

- Enter personal contact information
- Select a job position
- Choose availability via radio buttons
- Submit a short message
- Receive visual feedback based on input state

The project focuses on **semantic HTML**, **form validation**, and **CSS pseudo-classes** rather than JavaScript.
