# ForgeBottle – Product Landing Page

A responsive product landing page built for the FreeCodeCamp Responsive Web Design Certification.

---

## 📌 Project Overview

ForgeBottle is a modern, responsive landing page showcasing a product with:

- Fixed navigation
- Smooth anchor scrolling
- Embedded product video
- Email submission form with HTML5 validation
- Flexbox-based layout
- Responsive design using media queries

This project fulfills all required FreeCodeCamp user stories.

---

## 🧱 Technologies Used

- HTML5 (Semantic Structure)
- CSS3
- Flexbox
- Media Queries
- Responsive Design Principles

## ✅ FreeCodeCamp Requirements Completed

✔ Header element with `id="header"`  
✔ Logo image with `id="header-img"` and valid external source  
✔ Navigation bar with `id="nav-bar"`  
✔ At least three `.nav-link` elements  
✔ Anchor links scroll to matching sections  
✔ Embedded video with `id="video"`  
✔ Form with `id="form"`  
✔ Email input with:
  - `id="email"`
  - `name="email"`
  - `type="email"`
  - `placeholder` text
✔ Submit input with:
  - `id="submit"`
  - `type="submit"`
✔ Form action set to:
  `https://www.freecodecamp.org/email-submit`
✔ Navigation fixed to top of viewport  
✔ Uses CSS Flexbox  
✔ Includes at least one media query

## 📱 Responsive Design

- Navigation stacks on smaller screens
- Form switches to vertical layout on mobile
- Feature and pricing cards wrap using Flexbox
- Video scales responsively with viewport width
